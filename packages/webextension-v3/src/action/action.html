Tutorial

The RecipeSage browser extension allows you to quickly snip recipes from the internet.
  1. Navigate to a page that contains a recipe.
  2. Activate the RecipeSage clip tool by clicking the RecipeSage icon in your browser's toolbar (you've already done this to view this popup).
  3. The clip tool will automatically grab the contents of the page and find recipe content within it.
  4. The saved recipe will open in a new tab once the process is complete.
To get started, click the dismiss button below and then start with step 1.
